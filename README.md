# README

* Ruby version : 2.6.x

* Rails version : 6.x

* System dependencies : runs on mac and linux

* Configuration : no config needed

* Database creation : run migration to create and use db (uses sqllite) 

* How to run the test suite : ($ rails s)

It is a really simple project. Just clone the repository and if you have all dependencies use $ rails s in app directory to run the code. Then you can observe results in localhost.
